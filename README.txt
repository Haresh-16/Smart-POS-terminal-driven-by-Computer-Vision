1. Install Python version 3.6.7

2. In CMD , pip install requirements.txt ( requirements.txt is present in the provided folder)

3. Run SmartPOSFinal.py , in the provided folder ,in the virtual environment where
   all dependencies mentioned in requirements.txt are installed in.

4. Connect Barcode scanner to COM7 and Arduino to COM5.

5. USB Camera can be connected to any of the remaining USB ports.
